Én Visual Studio Code-ban csináltam a dolgokat, szóval csak ott tudom neked elmagyarázni, hogy mit kell.
Ha le van töltve a python, és a VS:Code-ba is a python (ha nem tudod, hogy kell telepiteni Code-ba akkor nézz róla videót, nehéz itt elmagyarázni)
Ha minden megvan írd be a consoleba ezeket:
pip install flask
pip install flask-sqlalchemy
pip install flask-bcrypt
pip install flask-login
pip install flask-wtf
pip install wtforms

Asszem ennyi. 
Most még nem csináltam meg, hogy külön file-ba mentse a dolgokat, szóval külön file adatbázis.
Most még csak a RAM-ba menti le a memory-ba (confi.py fileba meg is tudod nézni)
Ha el akarod indítani a run.py filet indítsd el, majd menj fel arra az IP címre amit kiír, de gondolom tudod.

Nyílván még lehet bugos az oldal, ha találsz benne, akkor szólj, meg lehetőleg a hibakódot is írd le.
Remélem jó lett! :D
